## 结构：
- Logger

- Appender 类被看作为输出的目的地
  - encoder
    - pattern
  - 实现类
    - console
    - files
    - Syslog
    - TCP Sockets
    - JMS


- Layouts

- 层级关系：
  - root logger 作为 logger 层次结构的最高层
    - org.slf4j.Logger.ROOT_LOGGER_NAME
  - com.foo 的 logger 是名为 com.foo.Bar 的 logger 的父级

- 等级：
  - TRACE
  - DEBUG
  - INFO
  - WARN
  - ERROR


## 配置：
### 作用域：
